# Agent Memory

**A simple, local-first memory tool for developers and AI agents.**

Never lose an idea again. Store, search, recall, and organize your thoughts, notes, brainstorming sessions, and dev pain points — everything stays private on your device.

### Features
- Persistent memory that never forgets
- Tag-based organization (`#gen0-ai`, `#meridian`, `#photon-conductor`, etc.)
- Fast search across all memories
- Auto-fetch trending dev pain points from Hacker News and Reddit
- Dynamic tool suggestions based on logged pain points
- Auto-summarize trends
- Edit / delete memories
- Export to clean GitHub-ready markdown
- Runs on Termux, laptops, or any Python environment
